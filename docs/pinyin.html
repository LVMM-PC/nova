    Pinyin

    引用

    • http://pic.lvmama.com/min/index.php?f=/js/lv/pinyin.js

    调用方式

                
            var py = novaPinyin();

            //1.获取汉字拼音，默认以空格分隔
            py.getPinyin("我知道那些夏天就像青春一样回不来");
            // wo zhi dao na xie xia tian jiu xiang qing chun yi yang hui bu lai

            //2.获取汉字拼音，不分隔
            py.getPinyin("我知道这个世界每天都有太多遗憾");
            // wozhidaozhegeshijiemeitiandouyoutaiduoyihan

            //3.获取汉字拼音，以"-"分隔
            py.getPinyin("所以你好再见");
            // suo-yi-ni-hao-zai-jian

            //4.获取姓名拼音
            py.getNamePinyin("令狐冲");
            // { lastName: "ling hu", firstName: "chong" }

    实例

    汉字转拼音

    suo yi ni hao zai jian

    姓名转拼音

    ling hu
