# nova

1. 安装node.js，https://nodejs.org/en/
2. 开启命令行，切换至项目目录
3. 安装npm包，npm i
4. 全局安装gulp，npm i -g gulp
5. 编译sass文件，gulp sass
6. 监控sass文件变更，gulp sass:watch
